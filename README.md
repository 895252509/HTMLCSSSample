重新学习HTML5和CSS3的代码示例

1.   `<!--...-->	`定义注释。
    * 定义和用法
        ```html
        <script type="text/javascript">
        <!--
        function displayMsg()
        {
            alert("Hello World!")
        }
        //-->
        </script>
        ```
    * 属性
        无
    * 事件
        无
